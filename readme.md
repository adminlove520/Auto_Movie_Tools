# Auto_movie_Tools

爬取豆瓣下月即将上映电影并生成到微信公众号图文文章

1. `.env` 填写信息
2. 安装依赖:

```
matplotlib
werobot
dotenv
requests
webptools
bs4
```

3. `python3 main.py`
